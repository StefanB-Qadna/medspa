## Design system

### Added — `src/app/design-tokens.css` (244 lines, the single source of truth)

Full token system, organized into commented sections:

- **Color** — brand (cream, linen, warm-dark, warm-darker, brass + variants, forest), neutral/border (border, border-strong, placeholder), state (error, error-bg, success, success-bg, warning), overlay/scrim (warm-dark-tinted, never pure black).
- **Typography** — `--font-serif` and `--font-sans` with full fallback chains; fluid `clamp()`-based size scale `--text-xs` → `--text-display`, each with a paired `--text-*--line-height`; weight, tracking (incl. `--tracking-label` for eyebrows and `--tracking-logo` for the wordmark), leading.
- **Spacing** — `--spacing` base unit drives every Tailwind `p-*` / `m-*` / `gap-*` / `size-*`.
- **Radius** — `none`, `xs`, `sm` (4 px), `md` (8 px), `lg` (16 px), `xl` (24 px), `2xl` (32 px), `full`. Replaces the old conflated `md == lg == 12px`.
- **Shadow** — `xs` → `2xl`, all tinted warm-dark for brand cohesion. Plus `--shadow-brass-sm/-md` for warm CTAs.
- **Easing** — exponential curves (`standard`, `out`, `out-quart`, `out-expo`, `in`, `in-out`). No bounce / elastic.
- **Breakpoints** — standard 5 (`sm` 640 → `2xl` 1536).
- **Motion duration** — `instant` / `fast` / `base` / `slow` / `slower` / `slowest`. Auto-collapse to `0ms` under `prefers-reduced-motion`.
- **Z-index scale** — `base` through `tooltip`.

### Modified — `src/app/globals.css`

- Imports `design-tokens.css` at top.
- All hex literals replaced with `var(--color-*)` references.
- Removed `scroll-behavior: smooth` from `html` (Lenis owns scroll).
- Added `.skip-to-content` styles for the new accessibility skip link.
- Unified prose `line-height` with body (`1.7`).
- `prose-medspa .italic` now uses `color-mix()` at 80% (was 60% — too faint to read as emphasis).
- `transition` on `a, button` now uses motion + easing tokens.

### Tailwind v4 note for reviewers

Tokens are tree-shaken — only utilities you actually use produce CSS variables in the bundle. `--shadow-md`, `--text-display`, etc. only land in the served CSS once something references them. This is expected v4 behavior.

- **Trust-badge row deleted entirely.** 5 icon+label badges + the 64-line `trustBadges` data array gone. Their content is either better conveyed by the Dr. Robledo section (surgeon-led care, credentials) or redundant (5-star reviews, "clean modern clinic").
